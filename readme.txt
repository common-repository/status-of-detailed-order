WooCommerce sipariş ekranını daha güzel göstermeye yarar. Woocommerce order status to make it look better.

== Description ==

Bu eklenti WooCommerce eklentisini kullanan websitelerde kullanıcı panelinde bulunan sipariş durumu bölümünü düzenlemeye daha güzel göstermeye yarar. 

== Installation ==

1. Eklenti dosyalarını 'wp-content/plugins' dizinine yükleyin veya eklentiyi doğrudan WordPress eklentileri ekranından yükleyin.
2. Eklentiyi WordPress'teki 'Eklentiler' ekranından etkinleştirin.
3. Eklentiyi kurmak bu kadar basit !
